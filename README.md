# cdate
A dead simple command-line calendar

# Installation
- Clone the repo
```bash
cd cdate
make fclean
vim config.h # Edit the config file to your liking, and compile !
make
make clean
sudo cp cdate /usr/bin # or whatever your bin path is !
```

Then `cdate` to use it !
